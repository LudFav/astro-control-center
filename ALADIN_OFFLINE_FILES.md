# Fichiers Aladin Lite a telecharger (offline)

Pour remplacer D3-Celestial par Aladin Lite sans CDN, place ces fichiers a la racine du projet (au meme niveau que `index.html`):

- `aladin.min.js`
- `aladin.min.css`

Sources officielles:

- Package complet v3 (contient les assets):  
  https://aladin.cds.unistra.fr/AladinLite/api/v3/latest/aladin-lite.tgz
- Documentation release:  
  https://aladin.cds.unistra.fr/AladinLite/doc/release/
- GitHub:  
  https://github.com/cds-astro/aladin-lite

## Structure attendue

- `index.html`
- `aladin.min.js`
- `aladin.min.css`

## Verification rapide

Si les fichiers manquent, la carte affichera:

- `Aladin Lite indisponible`

et l'app restera fonctionnelle pour le reste (liste objets, GoTo, controle monture).
